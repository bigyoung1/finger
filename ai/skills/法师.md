# 法师攻略

## 核心目标
凑出 0，然后打 [0,1/5/8/9]：80物+45法+1层雷霆。HP160极低，靠雷霆回血续命。

## 优先级
1. **有 0 时必打人**：[0,1/5/8/9] = 80物+45法+雷霆，绝不回血
2. **凑 0**：让自己某只手变 0（下回合必打组合）
3. **叠雷霆**：给多个敌人叠雷霆，回合末按偶数个数触发
4. **绝不凑 [2,2]/[3,3]**：30护盾对法师没意义，浪费回合

## vs 各角色
- vs 藏师：优先打其他目标，藏师物伤减半+反弹，打了亏
- vs 孙悟空：保持自己手为奇数（避免雷霆触发），用法伤压（不更新x）
- vs 忍者：小心对方手上7，中毒会消耗法师宝贵HP
- vs 大乔：速杀（HP120最低），大乔死后无人抢血

## 手牌管理
- 优先让自己双手都是奇数（1/3/5/7/9）→ 对方雷霆不触发
- 凑 0 最佳路径：用 1 碰对方 9，或用 3 碰对方 7
