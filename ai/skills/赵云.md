## 核心定位
攻防联动型半肉：x 随回血增长，y 随输出增长，越打越强的雪球角色。
单手 1/4 被动回血，单手 5/8/9 被动物伤，每次攻防行为都在强化下一次。

## 核心机制
- **x（物伤加成）**：初始20，每次回血后 = 本次总回血量（越回血，下次打得越狠）
- **y（回血加成）**：初始10，每次物伤后 = 本次总输出/2（越打人，下次回得越多）
- **calculateOutputDamage**：物伤 + x，双4/鸦眼buff等乘数作用在含x的总值上
- **calculateFinalHeal**：回血 + y，坦克加成也乘在总值上
- **单手1/4被动**：新变出时回复10血（触发 y 加成，即实际回 10+y）
- **单手5/8/9被动**：新变出时造成20物理伤害（触发 x 加成，即实际造 20+x）

## 优先级
1. **[0,1/5/8/9]**：40物伤触发 x 加成，同时 y 随本次输出/2提升
2. **[0,4/6]**：回30血触发 y 加成，同时 x 随本次总回血提升
3. **凑单手 1/4**：被动回血 10+y，更新 x = 总回血
4. **凑单手 5/8/9**：被动物伤 20+x，更新 y = 总输出/2
5. **[6,6]**：回90+y血，大幅提升 x
6. **[9,9]**：爆发输出，触发 x 合并后再乘 1.5^N

## 战术要点
- 开局 x=20/y=10 较弱，前期尽量回血（凑1/4/6组合），提升 x 让后期打伤更高
- x 和 y 相互强化：回血→提升x→打伤更高→y提升→回血更多→循环
- 注意单手被动在防套娃保护内不会再次触发循环，安全稳定
- 2v2坦克加成：calculateFinalHeal 含坦克加成后再 +y，叠乘效果强

## 克制关系
- 强对：前中期打藏师（消耗藏师HP让赵云 x 积累），后期爆发高
- 弱对：大乔（频繁回血提升了赵云 x，但大乔会把 RECOVERY 抢走，y 提升受阻）
