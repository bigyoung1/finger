⚡ 基础规则

🖐️ 每回合做什么？

1
选自己的一只手（左手或右手）
2
点对方一只手（数字不为 0）
3
两数相加取余 10，结果更新到自己选的那只手
我左手 3，碰对方右手 5 → 3+5=8 → 我左手变 8
我右手 7，碰对方左手 6 → 7+6=13 → 3 → 我右手变 3
不能碰对方数字为 0 的手

🎯 触发技能的时机

每次行动结束后，系统自动检查你的双手组合，满足条件就立即触发对应效果。 同一个效果只在新变出来时触发一次，不会重复计算。

【6,6】大回血

【0,0】真实伤害

✨ 单手特效——单独变出时触发

单手出现 6 → 回复 30 血
单手出现 7（忍者专属） → 给对方 1 层中毒
单手出现 2/3（小乔专属） → 获得 20 点护盾
⏳ 0 的特殊规则
⚠️ 0 有倒计时！
一只手变为 0 后，开始 2 回合倒计时。倒计时归零时，必须用那只 0 的手行动（系统强制）。
被碰出新数字就解除倒计时。
⚰️
双手全为 0 → 立即死亡，要小心同时让两只手都变 0 的情况
💡
对方双手全为 0 时，你本回合跳过，但 buff / 护盾 / 中毒照常结算
🎯
主动"套 0"是重要战术——让对方被迫在不好的时机出手
⚔️ 伤害类型
🗡️ 物理伤害
可被物理盾抵挡
✨ 法术伤害
可被法术盾抵挡
💀 真实伤害
穿透所有护盾，无敌也无效
☠️ 毒伤（法术）
中毒每层每回合扣 10

护盾也分类型：物理盾只挡物伤，法术盾只挡法伤，物法盾两种都挡，真实盾挡真伤。 护盾剩余回合短的优先消耗，每回合流逝一次。

💊 状态系统

☠️ 中毒

每层中毒在回合结束时造成 10 点法术伤害。 用 回复（RECOVERY）类回血 可以解毒：每 20 点回血消解 1 层毒。

🛡️ 反弹盾

持有反弹盾时，受到物理伤害会将等量伤害反弹给攻击者（反弹不会杀死对方，最多打到 1 血）。

🧊 冰冻

下一个自己的行动回合完全跳过，但中毒、护盾、倒计时照常结算。

✨ 无敌

免疫物理和法术伤害，真实伤害仍然穿透。

👥 2v2 对战规则

🏟️ 行动顺序

行动顺序为 1号 → 2号 → 3号 → 4号，循环进行。 1号&3号为 HERO 队，2号&4号为 REBEL 队，交替行动。 只能碰敌人，不能碰队友。

👥 双半肉流

  • 每回合可自由切换抗伤位
  • 对方只能打到当前抗伤位
  • 抗伤位获得额外回血加成和护盾加强

⚔️ 坦脆流

  • 指定一人为坦克，另一人为脆皮
  • 脆皮只能打对方坦克
  • 坦克可自由选择攻击目标
  • 坦克获得永久回血和护盾加强

🤝 帮抗机制

队友即将被一击打死时，弹窗询问你是否帮抗。 帮抗后你承受本次全部伤害 × 1.5，但队友完全不受伤。 如果帮抗后你也会死，弹窗不出现。倒计时 10 秒，超时视为放弃。

💡 开局建议

1
先尝试凑出 0 组合（碰出 0 后立刻利用），收益最稳定
2
注意 0 的倒计时，不要让两只手同时变 0
3
血量低时优先回血（[0,4] / [0,6] / [x,6]），血厚时集中进攻
4
看懂对方的手牌，阻止对方凑双子星或 0 组合
